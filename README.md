# ChromeExtensions
_Como pentester web, siempre sentí la necesidad de tener herramientas cotidianas a mano. Por eso, se me ocurrió crear una serie de extensiones para Google Chrome que pudieran ser útiles para cualquier profesional, ya sea WebDev, Bug Bounty Hunter, Pentester, etc._

## Extensiones disponibles
